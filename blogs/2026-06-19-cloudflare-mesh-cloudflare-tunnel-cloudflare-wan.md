The Routes dashboard page now consolidates routes across connectors (Cloudflare Mesh, Cloudflare Tunnel, Cloudflare WAN, and Magic Transit) into a single table. Users can visualize networks with an interactive map, manage routes without leaving the page, and test routes before committing changes.
