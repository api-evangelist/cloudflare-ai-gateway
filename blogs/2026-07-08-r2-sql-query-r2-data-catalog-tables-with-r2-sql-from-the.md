You can now query your R2 Data Catalog tables with R2 SQL directly from the Cloudflare dashboard, without installing a CLI or wiring up a client. This makes it easy to explore your Apache Iceberg data, validate queries, and inspect results in one place. To get started, go to R2 Data Catalog in the Cloudflare dashboard and select Query data to launch the built-in SQL editor.
